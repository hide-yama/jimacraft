# 販売方法

## 状態: 決定済み

## 検討すべきポイント
- [x] 自社サイト直販 vs SaaSプラットフォーム（Gumroad等）vs 両方
- [x] SaaS型（ブラウザで使う）vs ダウンロード型 vs ハイブリッド
- [x] デモ・無料版の提供方法

## 決定事項
- **SaaS型（ブラウザでアクセス）** で提供する
- 自社サイト直販（Stripe決済）
- **フリーミアム**: 無料プラン（機能制限あり）+ 有料Proプラン
- 無料プランが体験・集客の導線になる設計

### 選定理由
- 現在の構成（HTML + Express サーバーAPI）がそのまま活かせる
- ターゲット（ディレクター等）にとってURLを開くだけで使えるのが最も簡単
- ダウンロード型はNode.js環境構築が必要で非技術者には敷居が高い
