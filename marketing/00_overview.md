# jimacraft マーケティング全体マップ

## 現状

### プロダクト概要
- 文字起こしテキストからSRT字幕ファイルを生成するWebアプリ
- 動画ファイル → 文字起こし → SRT変換 → 話者別分割 → Premiere Pro インポートまでを一貫サポート
- **2〜3時間の字幕作業を約10分に短縮**

### 開発の経緯
テレビディレクターの知人が、動画の字幕付け（話者ごとの色・スタイル変更含む）に大きな手間をかけていた。その課題を解決するために開発。テレビに限らずYouTube等の動画編集全般に需要があると考え、販売を検討中。

### 技術構成（現状）
- フロントエンド: HTML + JavaScript（単一HTMLファイル）
- バックエンド: Node.js + Express（SRT変換・編集・話者分割のAPI）
- 文字起こし: ユーザー自身が Google AI Studio で実施（アプリはプロンプト生成を支援）
- MP4→MP3変換: ユーザーがローカルで ffmpeg コマンドを手動実行（アプリはマニュアルを表示）
- 現在は localhost で動作（デプロイ先は未定）

### ターゲットユーザー（想定）
- テレビ番組のディレクター・編集者
- YouTuber / 動画クリエイター
- 映像制作会社のスタッフ
- Premiere Pro を使って字幕を付ける人全般

## 決定事項ログ

1. **課金システム**: Stripe を使用（サブスク型）
2. **販売方法**: SaaS型（ブラウザアクセス）、自社サイト直販、フリーミアム
3. **料金設定**: Free(¥0) / Plus(¥480/月) / Pro(¥980/月) の3段階。機能・話者数・回数で差別化
